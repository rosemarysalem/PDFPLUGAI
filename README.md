📚 PDF Plug AI - AI-Powered PDF Learning Tool

**Transform PDFs into Interactive Learning Experiences with AI-powered summaries, quizzes, flashcards, and comprehension tools. Smart PDF processing for enhanced learning.**

PDF Plug AI is a powerful Chrome extension and web application that converts any PDF document into an interactive learning platform using advanced AI technology. Using smart PDF processing and comprehension, whether you're studying textbooks, research papers, or any document, PDF Plug AI helps you learn more effectively through multiple AI-powered study modes.

## 🚀 Key Features

### 🧠 **Document Analysis**
- Deep comprehension and key insights extraction
- Adjustable analysis depth (Basic, Intermediate, Advanced)
- Context-aware responses based on document content

### 🎯 **Interactive Quizzes**
- AI-generated multiple choice questions
- Real-time scoring and progress tracking
- Adaptive difficulty based on document complexity

### 📚 **Smart Flashcards**
- Interactive, flippable flashcard interface
- Up to 10 cards per session for intensive study
- Card controls: Reset, Show All Answers, Shuffle
- Perfect for textbook page-by-page studying

### 📝 **Smart Summary**
- Multiple summary types:
  - **Comprehensive**: Complete document overview
  - **Section-by-Section**: Organized by document structure
  - **Executive**: Concise key points summary
  - **Detailed Analysis**: In-depth insights and implications
- Automatic summary generation for uploaded documents
- Professional formatting for easy reading

## 🛠 Technical Specifications

### **AI Integration**
- **OpenAI API**: GPT models for high-quality responses
- **OpenRouter API**: Meta Llama models for alternative AI processing
- Seamless switching between AI providers
- Optimized prompts for educational content

### **PDF Processing**
- PDF.js integration for client-side processing
- Text extraction and page-by-page navigation
- Support for complex document structures
- Web PDF capture from browser tabs
- **URL Loading**: Direct PDF processing from web links
- CORS proxy support for cross-origin PDF access

### **User Interface**
- Modern, responsive design with dark theme
- Mobile-friendly interface
- Accessibility features and keyboard navigation
- Real-time loading indicators and error handling

## 📋 Prerequisites

1. **API Key**: OpenAI API key or OpenRouter API key
2. **Browser**: Chrome browser (for extension functionality)
3. **Internet**: Active internet connection for AI processing

## 🔧 Installation & Setup

### **Method 1: Chrome Extension**
1. Download the extension files
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extension folder
5. The PDF Plug AI icon will appear in your toolbar

### **Method 2: Web Application**
1. Clone or download the project files
2. Open `dashboard.html` in your web browser
3. No additional installation required

## 🎯 Quick Start Guide

### **Step 1: Configure API**
1. Select your AI service (OpenAI or OpenRouter)
2. Enter your API key in the configuration field
3. The key is securely stored locally

### **Step 2: Load Document**
**Option A: Upload Local File**
1. Click "📄 Upload & Parse PDF"
2. Select your PDF file from your device
3. Wait for processing completion

**Option B: Load from Web URL**
1. Enter a direct PDF URL in the URL input field
2. Click "🌐 Load from URL"
3. Wait for the PDF to be fetched and processed

*Note: URL loading works with direct PDF links. Some websites may have CORS restrictions.*

### **Step 3: Choose Study Mode**
1. Click any feature card or use mode buttons:
   - 🧠 **Document Analysis**: Ask specific questions
   - 🎯 **Interactive Quiz**: Test your knowledge
   - 📚 **Smart Flashcards**: Memorize key concepts
   - 📝 **Smart Summary**: Get document overviews

### **Step 4: Start Learning**
- All features work automatically after PDF upload
- Adjust comprehension level as needed
- Export results for future reference

## 🐛 Troubleshooting

### **Common Issues**

**"Please enter your API key"**
- Ensure you've entered a valid OpenAI or OpenRouter API key
- Check that the correct service is selected

**"Please upload a PDF first"**
- Upload a PDF document before using study features
- Ensure the PDF was successfully processed

**"Rate limit exceeded"**
- Wait a few moments before making another request
- Consider switching to OpenRouter for different rate limits

**Buttons not working**
- Refresh the page and try again
- Ensure JavaScript is enabled in your browser
- Check browser console for error messages

**"Unable to load PDF from URL"**
- Ensure the URL points directly to a PDF file
- Check if the website allows cross-origin requests
- Try downloading the PDF and uploading it instead
- Verify the URL is accessible and not behind authentication
